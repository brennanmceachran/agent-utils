**What this gives you**
- A single tool call that returns a summary, key caveats, and source URLs.
- Preset depth controls (fast, balanced, deep).

**Try it**
```yaml
tool: webresearch
args:
  reason: "Decide whether to adopt Next.js MCP"
  query: "Next.js MCP guide; next-devtools-mcp repo"
  quality: balanced
```
