**Why teams use it**
- Keeps long tasks moving without re-prompting every step.
- Reduces drift by capping iterations and forcing explicit completion.
- Creates a durable progress log for handoffs.

**What ships with it**
- An OpenCode agent, command, and plugin that run the loop.
- Prompt file updates for progress, verification, and next steps.
- Built-in completion token and a manual stop control.
